# EzLinux

EzLinux - Currently for Debian Based (APT Based) Machines only. Aids in joining things like Active Directory, Creating automatic updates and making the terminal a pretty colour! 

Looking to make versions for arch/centos/RHLE

<p align="center"><img width="80%" src="./images/ezlinux-updated.png" /></a></p>

<br />

# Install

`wget https://git.io/JDuBu -O EzLinux && chmod +x EzLinux && sudo ./EzLinux`

# Usage
```
Arrow keys to navigate :arrow_up: :arrow_down: :arrow_left: :arrow_right:
```
# Features
:heavy_exclamation_mark: Dynamic Active Directory Usage - You can set the domain, AD Admin and OU directly from EzLinux


:heavy_exclamation_mark: Enable/Disable Root SSH, config files are a thing of the past, especially when bringing up multiple servers.

:heavy_exclamation_mark: Coloured Syntax can be enabled directly to the global .bashrc
:heavy_exclamation_mark: Automatic updates, along with logfiles at /var/log/apt/Automaticupdates - cronjob auto sets-up for Thursday, 1am. :clock1:
:heavy_exclamation_mark: Custom Tailor To Your Own needs!
